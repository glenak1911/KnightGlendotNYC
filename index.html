About

Hi, Glen here! I'm a web developer and programmer located in The Greatest City in the World, New York City. I first began working with HTML during elementary school when I purchased an instructional book from a school book fair. I started when all styling was done within the html, that's right, before CSS was even a thing! My html files were even stored on a floppy disk with a whopping 5mb of memory.

Fast forward a few years and I majored in Information Technology while studying at Florida International University. During that time I picked up Java, and dabbled in Visual Basic, C#, and JSP/Servlet technologies, among the other core IT courses. I enjoyed the Java courses I took the most, as well as my web development courses.

Now that I'm out of school, I've picked up Javascript and JQuery, and have been sharpening my CSS. I'm also working on learning how to use the different Javascript libraries(node.js, ember.js, Angular.js, etc), and picking up any other cerifications that I can pursue.
